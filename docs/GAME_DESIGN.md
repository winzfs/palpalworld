# PalPalWorld Game Design

## 한 줄 콘셉트

브라우저에서 친구들과 접속해 몬스터를 포획하고, 거점을 건설하고, 몬스터 자동화 생산 라인을 키우며, 던전과 보스를 공략하는 2D 멀티플레이 수집 생존 RPG.

## 핵심 플레이 루프

```txt
탐험 → 채집 → 전투 → 포획 → 거점 배치 → 자동 생산 → 제작/강화 → 더 위험한 지역 탐험
```

## 주요 시스템

### 1. 탐험

- 2D 탑다운 타일맵
- 초원, 숲, 광산, 사막, 설원, 화산, 고대 유적 지역
- 지역별 자원, 몬스터, 환경 위험 존재

### 2. 몬스터

몬스터는 세 가지 역할을 가진다.

- 전투 동료
- 거점 노동자
- 자동 생산 유닛

기본 속성:

- 무속성
- 불
- 물
- 풀
- 전기
- 얼음
- 어둠

작업 적성:

- 벌목
- 채광
- 운반
- 농사
- 제작
- 요리
- 발전
- 냉각
- 방어
- 탐색
- 치료

### 3. 포획

포획 확률은 다음 요소로 계산한다.

- 몬스터 현재 HP 비율
- 몬스터 희귀도
- 상태이상
- 포획 도구 등급
- 플레이어 포획 숙련도
- 속성 보너스

### 4. 거점

플레이어는 거점 코어를 설치하고 건물을 배치한다.

기본 건물:

- 거점 코어
- 작업대
- 보관함
- 침대
- 농장
- 제련소
- 요리대
- 몬스터 숙소

거점의 핵심 재미는 몬스터 작업 배치와 생산 라인 최적화다.

예시:

```txt
채광 몬스터 → 광석 생산
운반 몬스터 → 창고 이동
제작 몬스터 → 금속 부품 제작
건축 몬스터 → 건물 업그레이드
```

### 5. 전투

- 플레이어 직접 이동/공격
- 몬스터 AI 자동 전투
- 모바일에서는 가상 조이스틱 + 큰 버튼 UI 사용
- PC에서는 WASD + 마우스/키보드 조작 사용

### 6. 멀티플레이

- 같은 필드에서 플레이어 위치 동기화
- 채팅
- 파티 사냥
- 공동 보스
- 길드 거점
- 거래소

## MVP 범위

1차 MVP는 다음만 집중한다.

- 로그인 없이 임시 닉네임 접속
- 2D 필드 이동
- 실시간 위치 동기화
- 채팅
- 자원 채집
- 기본 몬스터 스폰
- 전투/HP
- 포획
- 인벤토리
- 거점 코어 설치
- 건물 배치
- 몬스터 작업 배치
- 간단한 제작
