# PalPalWorld

웹 기반 2D 멀티플레이 생존·수집·거점 자동화 RPG 프로젝트입니다.

팰월드의 핵심 재미인 생물 포획, 거점 운영, 자동 생산, 탐험, 전투를 모티브로 하되, 웹과 모바일에서 접근하기 좋은 2D 멀티플레이 게임으로 재해석합니다.

## 목표

- 설치 없이 브라우저에서 실행
- PC/모바일 반응형 UI 지원
- 실시간 멀티플레이 기반
- 몬스터 포획, 전투, 거점, 제작, 자동화 시스템 확장 가능
- 유지보수가 쉬운 모듈형 monorepo 구조

## 기술 구조

```txt
apps/
  web/              Next.js 기반 웹 클라이언트
  realtime-server/  Socket.IO 기반 실시간 게임 서버

packages/
  shared/           클라이언트/서버 공용 타입, 상수, 프로토콜
  game-core/        순수 게임 규칙, 계산식, 시뮬레이션 로직

docs/               기획, 아키텍처, 로드맵 문서
```

## 개발 원칙

1. 게임 규칙은 `packages/game-core`에 둔다.
2. 클라이언트와 서버가 같이 쓰는 타입은 `packages/shared`에 둔다.
3. 화면 UI와 렌더링은 `apps/web`에 둔다.
4. 실시간 통신, 방 관리, 위치 동기화는 `apps/realtime-server`에 둔다.
5. 모바일 조작을 처음부터 고려한다.

## 로컬 실행 예정 명령어

```bash
pnpm install
pnpm dev
```

현재는 초기 아키텍처 스캐폴드 단계입니다.
